# NailIt Business Input

NailIt is an AI interview intelligence product for candidates preparing for a specific interview at a named company.

The user provides:

1. CV
2. Job description
3. Target company
4. Relevant experience
5. Interview stage if known

The system should produce:

1. Company specific research
2. Role decoding
3. Candidate evidence mapping
4. Gap analysis
5. Prep pack
6. Answer guidance
7. Mock interview feedback

This is input only.

It is not evidence that people will pay.
